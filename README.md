                                                                  CGPA Calculator

A simple C++ program that helps students calculate their GPA and CGPA based on course grades and credit hours.
                                                                  
                                                                   Problem Solved

Manually calculating GPA and CGPA can be time-consuming and prone to errors. This program automates the process by calculating:

1-Total Credits
2-Total Grade Points
3-Semester GPA
4-Overall CGPA

                                                                    Features

1-Enter multiple courses
2-Input grades and credit hours
3-Calculate GPA and CGPA automatically
4-Display course-wise details and final results


How to Run

```bash
g++ CGPA.cpp-o CGPA
./CGPA
```
